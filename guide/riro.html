3개인정보 수정 (필수)

비밀번호를 변경하지 않으면 수강신청이 불가능합니다

개인정보수정 화면에서 빨간 표시 항목을 모두 입력하세요

  • ✓비밀번호 변경 — 영문·숫자·특수문자 모두 포함, 8자 이상 (최종 수강신청에도 사용되니 꼭 기억해두세요)
  • ✓이메일 필수 입력 (비밀번호 분실 시 이 메일로 재발급)
  • ✓진로 / 희망전공 필수 입력 (‘없음’으로 작성 가능)
  • ✓전화번호 확인 — 본인 · 어머니 · 아버지 번호 (추후 학부모 가입 시 필요)

‘비밀번호 변경’ 클릭 → 현재 비밀번호(초기값 123456) 입력 후 새 비밀번호 설정
